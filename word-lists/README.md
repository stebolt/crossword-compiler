# Word Lists

Place word list files here. Expected format: one word per line, plain text.

- `english.txt` — general English dictionary (UKACD or equivalent)
- `cultural.txt` — proper nouns: famous people, footballers, place names, etc.

These are loaded at runtime by the compiler's auto-fill engine.
Word lists are gitignored if large. Add your own or source from UKACD.
